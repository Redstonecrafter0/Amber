BaseModule

abstract class BaseModule(displayName: String, description: String, category: Category)
Content copied to clipboard

The base class for all modules.

Available settings:

  • button

  • switch

  • range

  • colorPicker

  • dropDownMenu

  • list

  • customSetting

Adding events:

object YourModule: BaseModule(...) {
    init {
        on<Event> { // Respects super classes
            // Do something
        }
    }
}

Adding settings:

object YourModule: BaseModule(...) {
    val id by button("displayName") {
        // Do something
    }
}
